<h1>Одностраничный сайт суши-кафе</h1>
<p>Довольно интересный лэндинг в котором есть много интересных плагинов которые можно реализовать.</p>
<p><strong>Например</strong></p>
<ul>
  <li>Фиксированное меню</li>
  <li>Корзина</li>
  <li>Модальное окно с подробным описанием карточки товара и возможностью заказа</li>
</ul>
<p>Проблемы с которыми столкнулся:</p>
<p>Построение сеток, пока не определился с методом построения сеток, изначально хотел использовать стандартные свойства Flex<br>
justify-content: space-between/space-around и так далее, но в случае если использовать эти свойства при установке админки для сайта<br>
и изменении количества элементов в блоке, сетка будет ломаться, пока что решил использовать для отступов между элементами фиксированный margin<br>
также задумался об использовании Grid темболее я их уже изучал, кажется что они решают эту проблему, но возникает проблемы<br>
поддержки Grid браузерами в данный момент 95% браузер позволяют использовать Grid, когда Flex 99%.<br>
Узнавал в чатах по верстке как вообще люди устраняют подобные проблемы и люди делились различными методоами<br>
с отрицательными margin и padding у элементов, но как-то не очень хочется использовать подобное. Пока что решил верстать фикс + адаптив<br>
хотя это очень долго.
</p>
